# Tasks designadas ao backlog

- Criação de sistema de "Gostei/ Não gostei"  
  Permite que os users avaliem o conteúdo publicado

- Criar painel adm  
  Área para administradores gerenciarem users, posts, comunidades e configurações de conteúdos

- Criar função de adicionar amigos  
  Usuários podem conectar-se entre si

- Criar função de criar grupo  
  Permite que os usuários criem grupos fechados de conversa

- Criar aba de comunidade  
  Seção onde ficam os conteúdos postados publicamente

- Criar versão para mobile  
  Adaptar o sistema já existente para dispositivos móveis

- Modo escuro  
  Adicionar o modo escuro para maior conforto do user

- Sistema de notificações  
  Envia alertas sobre interações, mensagens e atualizações do app

- Ranking de ideias  
  Classifica as ideias com maior prestígio entre a comunidade

- Página de suporte  
  Área para ajuda e contato com o suporte

- Upload de fotos nos posts  
  Permite que os usuários possam colocar fotos nos posts para maior visualização do público

- Feed personalizado  
  Usuário pode personalizar o conteúdo dos posts que quer receber

- Implementação de sistema de denúncia  
  Função com a finalidade de denunciar usuários, posts e comentários maliciosos

- Acessibilidade  
  Adição de modo de leitura e áudio para déficits visuais

- Integração Login ao Beck-end
  Vincular o sistema de login ao sistema beck-end

- Implementar tela de criação de perfil
  Tela que exibe informações que o usuario deve inserir para criar o perfil

- Implementar sistema de avaliação
  Adicionar as funções de "Gostar e não gostar"

- Implementar Historico de Busca
  Toda busca fica salva na aba de busca (pode ser desativada)

- Implementar sistema de ordem
  O user pode ordenar a busca por qualquer ordem de sua preferencia

- implementar CRUD de ideias
  Introdução do CRUD das ideais (possibilita o user publicar, ler, editar e apagar posts)

- Implementar login e autenticação
  Usuário cria conta com e-mail e senha, o sistema valida

- Conectar Banco de dados
  O banco será conectado ao sistema beck-end

- Criar tela de Login
  Dev front-end cria uma página de login que possibilita o usuário realizar o login

- Desenvolver sistema de tags
  Integrado a aba de busca, com a finalidade de encontrar usuários e posts

- Definir requisitos funcionais do sistema
  A definição das obrigatoriedades do sistema, onde se constroi a indentridade e funcionamento base do sistema

- Definir arquiteruta do sistema
  Dev escolhe quais tecnologias mais fazem o perfil do projeto, e avalia com base nos requisitos já levantados

- Modelar banco de dados
  Definição de como os dados serão organizados

- Criar wireframes das telas principais
  Esboçar como seria o front-end das principais telas do software

- Implementar sistema de categorias
  Um facilitador de busca, o user pode se denominar com determinada profissão, facilitando a inscrição em comunidades

- Termos de uso e política de privacidade
  Termos que o user deve concordar para o funcionamento do app, todas as regras, políticas, estão alocadas nesse espaço

- Implementar criação de perfil
  User pode criar o perfil, colocando nome, foto e categoria
