Aqui estão explicadas as funções atribuidas para meu desenvolvimento


1. Implementar Histórico de Transações: uma função de backend e banco de dados, 
foi criada uma tabela no banco de dados chamada transacoes, ferramentas: SQL (PostgreSQL/MySQL) para a estrutura e node.js e python.

2. Desenvolver Sistema de Busca: é a ferramenta que o investidor vai usar para ver o que interessa para ele,
esta ainda no planejamento fazer uma rota de API que aceite um parâmetro de texto, o código fará uma consulta no banco usando o comando
LIKE
ferramentas: SQL.

3. Criar Tela de Login: esta sendo finalizada e será a porta de entrada do aplicativo.
ferramentas: HTML5 para a estrutura, CSS3 (Flexbox) para centralizar o formulário na tela e deixá-lo bonito, e
 JavaScript para capturar os dados digitados e enviar para a validação.

4. Termos de Uso e Política de Privacidade:  sendo uma parte delicada fpoi finalizada a funcionalidade de conteúdo estático.
criando uma página simples que exibe o texto legal. para facilitar a manutenção, o texto foi escrito em markdown e convertido para HTML.
ferramentas: um gerador de termos online (para a base jurídica) e HTML/CSS simples para a exibição no app.
5. Implementar Sistema de Categorias: sendo um dos pilares da ideia o investidor pode filtrar suas categorias rapidamente.
criado um conjunto de tags com categorias pré-definidas, sistema filtra as ideias que pertencem a categori a selecionada.
ferramentas: Frontend (React ou JS puro) para manipular a lista de ideias exibidas e CSS para estilizar as "Tags" de categoria.

6. Implementar Criação de Perfil: aqui com a ajuda do dev gustavo fizemos
o lugar pnde o usuário define se é dono de ideia" ou investidor.
* Gustavo  fez a parte do front: criou o formulário de cadastro com nome, foto, bio, linkedIn.
* Adilson fez a parte do backend: criei a lógica para salvar esses dados no banco e associá-los ao ID do usuário que acabou de logar.
frramentas: HTML Forms e JSON para o transporte dos dados entre a tela e o servidor.
